Exemples JavaScript

Que peut-on faire avec JavaScript ?

Modifier le contenu

On peut modifier le contenu d'une balise HTML (ce qui se situe entre la balise d'ouverture, et la balise de fermeture).

Modifier les attributs

On peut modifier les attributs d'une balise HTML. Ici, on change l'attribut src de la balise img.

Modifier le style CSS

On peut également modifier le style CSS d'un élément HTML ciblé.
